# Security

This is a fork of CumulusCI maintained by Clariti Cloud Inc.

## Reporting Security Issues

Please report any security issue to [oss@claritisoftware.com](mailto:oss@claritisoftware.com) as soon as it is discovered.

For issues affecting the upstream CumulusCI project, you may also report to [security@salesforce.com](mailto:security@salesforce.com).

## Dependencies

This library limits its runtime dependencies in order to reduce the total cost of ownership as much as possible, but all consumers should remain vigilant and have their security stakeholders review all third-party products (3PP) like this one and their dependencies.

## Supported Versions

Security updates are provided for the latest release version. We recommend always using the most recent version of clariti-cumulusci.
